REACTOR SIMULATOR

What is a Reactor Simulator?

We are looking to build a virtual nuclear reactor simulator using a simplified model that can educate high school and middle school students about the basics of how a CANDU reactor works. In addition, we will design a 3D printable model that interacts with the simulator.

A rendered version of the CANDU model built last year.

CNES plans on starting this project this spring! If you are interested, please join our discord and mention that you want to participate in this project.
